# Auto-scroll-browsing-tool
自动完成滑动浏览任务Automatically complete the scroll browsing task

# 使用说明
- 用于自动完成浏览任务，只会完成浏览15秒的任务。
- 执行之前需要在手机上开启USB调试，并允许点击、滑动操作，并连接到电脑。电脑上需要安装uiautomator2，cnocr等Python库后方可使用。
- 启动脚本前需要先将手机停留在“做任务赚体力”界面，然后在powershell下切换到脚本所在路径，执行 python .\auto_browers.py 启动脚本，脚本会自动读取手机屏幕并识别页面中的浏览15秒任务，智能点击，并进行滑动和等待。完成所有浏览任务后会自动退出，并显示总完成数。可通过ctrl+C中断脚本。
